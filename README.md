# Targeting glucocorticoid-induced CD20 activation in preclinical models of B-ALL

This repository contains the R scripts used to generate the single-cell RNA-seq figures in the manuscript submitted to *Nature*.

Raw count matrices are available on GEO under accession number [GSE309451].
The preprocessed Seurat object used as input for `figures_script.R` was generated from these files using the preprocessing
parameters described below. It is available upon reasonable request to the corresponding author.

## Preprocessing parameters

The preprocessed Seurat object (`All_merge1_integrated.RData`) was generated from the raw count matrices using **Seurat v5** with the following parameters:

- **Input**: 4 samples (Diagnosis Vehicle, Diagnosis CHEMO, Relapse Vehicle, Relapse CHEMO)
- **Quality control**: cells with `percent.mt >= 3%` or `nFeature_RNA >= 6500` were excluded
- **Normalization**: `NormalizeData()` — LogNormalize, scale factor = 10,000
- **Variable features**: `FindVariableFeatures()` — VST method, 3,000 features
- **Scaling**: `ScaleData()` — linear model
- **Dimensionality reduction**: `RunPCA()` — 50 PCs, seed = 42
- **Integration**: `IntegrateLayers()` — CCA method (`new.reduction = "integrated.cca"`)
- **Clustering**: `FindClusters()` — resolutions 0.6, 0.8, 1.0; resolution 0.6 retained
- **Cluster 12 removed**: identified as mouse cell contamination
- **UMAP and tSNE**: computed on `integrated.cca` reduction

## Gene signatures

All gene signatures used for UCell scoring are embedded directly in `figures_script.R`:

- **MRD up-regulated genes** — Ebinger et al., *Cancer Cell* 2016
- **MRD down-regulated genes** — Ebinger et al., *Cancer Cell* 2016
- **Cell-quiescence up** — Min et al., *PLoS Biology* 2019
- **Cell-quiescence down** — Min et al., *PLoS Biology* 2019
- **BCR signaling — KEGG** — KEGG_B_CELL_RECEPTOR_SIGNALING_PATHWAY
